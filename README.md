# Простая реализация TO-DO списка

### Используемые инструменты:
- React
- TypeScript
- Vite
- MUI
- Jest
- node `v20.18.2`

### Инициализация и запуск:
- `npm i` - установка зависимостей
- `npm run start` - запуск приложения
